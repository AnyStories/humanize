# humanize

A package for translating numbers and dates into a human readable format.

## Usage

```dart

import 'package:humanize/humanize.dart' as humanize;

main(){
    print(humanize.appNumber(1)); // one
    print(humanize.ordinal(3)); // 3rd
}
